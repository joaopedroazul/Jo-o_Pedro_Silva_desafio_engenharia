linguagem utilizada: Ruby
versão: 3.0.1
Sistema Operacional Utilizado: Ubuntu 20.04 LTS


Antes de rodar o programa peço que insira os dados no arquivo dados.txt
no mesmo padrão que lá se encontra.

Para rodar o codigo basta abrir um terminal na pasta
e rodar o seguinte codigo:

ruby desafio_Loggi.rb

Sugestões para rodar o programa
Ubuntu: instalar o ruby via RVM normalmente é mais facil
Windows: Baixar o RubyInstall no site oficial ou rodar no Ubuntu via WSL

Sobre a Lógica do programa, ela está resumida em comentarios ao longo do código
